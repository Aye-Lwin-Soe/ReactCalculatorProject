# ReactCalculatorProject

A simple calculator application built with React.js. This project demonstrates React fundamentals such as components, state management, event handling, and user interactions.

## Features

* Addition, Subtraction, Multiplication, and Division
* Clear and Delete functions
* Responsive user interface
* Real-time calculation display
* Built with React Hooks

## Technologies Used

* React.js
* JavaScript (ES6+)
* CSS3
* HTML5

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ReactCalculatorProject.git
cd ReactCalculatorProject
```

Install dependencies:

```bash
npm install
```

## Run the Application

Start the development server:

```bash
npm start
```

The application will be available at:

```text
http://localhost:3000
```

## Build for Production

```bash
npm run build
```
